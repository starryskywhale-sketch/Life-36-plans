# Life 36Ã—10 Planner

A simple offline HTML planner that splits a year into 36 blocks of 10 days.

## Features
- Daily check-in (tasks + mood)
- 36 sprints overview and goals
- Local storage backup/export & import

## How to use
1. Download `life36.html`
2. Open it in a browser (Chrome / Edge / Safari)
3. Your data is stored in the browser (localStorage)

## Notes
- Clearing browser data may remove your records. Use Export/Import to backup.
- Built with the help of AI tools. Provided as-is.
